OOP is a programming paradigm based on the concept of "objects", which can contain:

  • Data (attributes or properties)
  • Methods (functions that operate on the data)

Core principles of OOP:

  • Encapsulation – Bundling data and methods that operate on that data within one unit (object).
  • Abstraction – Hiding complex implementation details and showing only essential features.
  • Inheritance – Creating new classes from existing ones, inheriting attributes and behaviors.
  • Polymorphism – Allowing different classes to be treated as instances of the same class through a common interface.
